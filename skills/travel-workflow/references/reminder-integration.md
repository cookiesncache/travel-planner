# Reminder Integration

Set reminders for time-sensitive prep tasks (e.g. visa application, flight check-in). Use whichever reminder capability is available and best fits the user's context. Only set reminders the user confirms.

## Available reminder capabilities

Check which of the following are available in the current session:

- **Calendar** — create a calendar event or reminder alert on the relevant date
- **Task app** — create a task with a due date in the connected task app
- **Scheduled-task** — session-level capability to schedule a timed reminder (not a connected app). Check whether scheduled-task tools are available before attempting to use them. This capability is not always present.

If none are available, note in the plan that reminders would need a connected calendar or reminder tool.

## When no calendar is connected

If dated items have no calendar to sync to, record them in a "Key dates" section near the top of the plan artifact so they are easy to scan. Then offer reminders using whatever capability is available (task app, scheduled-task). If no reminder capability is available, note that reminders would need a connected calendar.
